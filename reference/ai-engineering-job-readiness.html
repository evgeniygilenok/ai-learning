Reference

AI engineering job readiness checklist.

Use this as the compressed reference for turning the roadmap into credible hiring signal.

Positioning

Senior TypeScript/product engineer moving into applied AI systems: RAG, evals, agents, observability, security, and production workflows.

Evidence Matrix

RequirementEvidence
LLM product integrationProduct triage assistant with structured outputs, streaming states, and validation.
RAGKnowledge assistant with chunks, metadata, citations, answerability, and retrieval evals.
EvaluationGolden datasets, CI smoke gate, before/after reports, and failure clusters.
ObservabilityTraces for retrieval, model calls, validation, tool calls, latency, cost, and failures.
AgentsControlled workflow with state, approvals, retries, recovery, and trace-based evals.
Tool safetyValidated, authorized, approval-gated, idempotent, and audited tool execution.
SecurityThreat model, redaction, ACL-aware retrieval, tenant boundaries, and prompt-injection mitigations.
Production readinessClean run path, seed data, CI, env config, health checks, and deployment notes.
Product judgmentUser feedback, adoption notes, scoped fixes, and evals derived from feedback.
Senior communicationArchitecture story, tradeoff memos, demo script, and interview answer bank.

Weekly Learning Loop

  • One lesson or reference pass.
  • One portfolio increment.
  • One eval, trace, or security improvement.
  • One documentation or interview-story update.

Weekly Job Loop

  • Save 5 roles.
  • Contact 3 people.
  • Run 1 interview drill.
  • Apply selectively after Stage 4, consistently after Stage 6.

Proof Package

  • 3-minute demo.
  • 10-minute architecture walkthrough.
  • Eval report.
  • Trace walkthrough.
  • Threat model.
  • Deployment notes.

Interview Drills

  • Debug a bad RAG answer from trace to root cause.
  • Design an eval for a new workflow.
  • Choose between RAG, fine-tuning, tools, or a workflow.
  • Threat-model an agent with write access.
  • Explain a cost/latency tradeoff.
  • Walk through a production incident involving model behavior drift.
